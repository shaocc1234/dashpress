## 关于
Dashpress 是一个管理应用生成器，可以帮助你快速高效地将数据库模式转换为时尚、可定制且高性能的管理仪表板。

### 安装

```bash
$ npx dashpress
```

就这样，你就可以在 http://localhost:3000 看到应用程序了。

如果你更喜欢使用 Docker，请查看仓库根目录中的 `Dockerfile.example` 文件。

### 演示
https://demo.dashpress.io 是一个使用 DashPress 生成的示例管理应用，无需编写任何代码。
